                    • 8. 总结

                        • 下面从优化目标出发，详细推导“基于类内散布矩阵的特征提取”为何要取最小特征值对应的特征向量作为投影方向，并由此得到降维变换矩阵。

                        1. 问题设定

                        设某个类 ω 的样本集为 {X1,X2,…,Xm}，每个样本是 n 维列向量。
                        该类内散布矩阵（协方差矩阵）定义为：

                        C=1m−1∑i=1m(Xi−μ)(Xi−μ)T

                        （无偏估计）或直接用最大似然估计（分母 m）。无论哪种，C 是 n×n 实对称半正定矩阵。

                        特征提取的目标：将 n 维样本 X 投影到 m 维空间（ m<n），得到 X∗=AX，其中 A 是 m×n 矩阵，其每一行是一个投影方向（单位向量）。通常要求各行相互正交。

                        对于基于类内散布矩阵的降维，我们希望投影后同类样本的类内方差尽可能小（即同类样本在投影空间中尽量聚集），从而有利于后续分类。这正是 最小化投影后方差 的准则。

                        2. 投影后方差的表达式

                        考虑单个投影方向 u （ n×1 单位列向量）。样本 X 在该方向上的投影为 uTX，是一个标量。投影后方差为：

                        Var(uTX)=uTCu

                        因为 C=Cov(X)，且 u 为确定性向量。

                        我们要选择 u 使得 uTCu 最小化，同时 uTu=1。

                        3. 转化为特征值问题

                        构造拉格朗日函数：

                        L(u,λ)=uTCu−λ(uTu−1)

                        对 u 求偏导并令为零：

                        ∂L∂u=2Cu−2λu=0⇒Cu=λu

                        这正是特征值方程。因此，投影方向 u 必须是 C 的特征向量，对应的拉格朗日乘子 λ 就是特征值。此时投影方差：

                        uTCu=uT(λu)=λ(uTu)=λ

                        所以投影方差等于对应的特征值 λ。

                        4. 最小化投影方差 ⇒ 选择最小特征值

                        因为我们要最小化投影方差，而方差等于特征值，所以应选择 最小特征值 对应的特征向量作为第一投影方向。

                        接下来，如果还需要第二个投影方向（与第一方向正交），我们在正交子空间中重复上述过程。由于 C 是实对称矩阵，不同特征值对应的特征向量自然正交，因此第二个投影方向就是次小特征值对应的特征向量。依此类推。

                        结论：要得到 m 维投影（降维到 m 维），应选择 C 的 m 个最小特征值对应的特征向量，将它们作为矩阵 A 的行（或列，取决于约定），构成线性变换 X∗=AX。

                        5. 为什么不是取最大特征值？与PCA的区别

                        • PCA（主成分分析）：最大化投影后方差（保留数据的主要变化方向），取最大特征值对应的特征向量。

                        • 基于类内散布矩阵的特征提取：最小化投影后方差（使同类样本尽可能聚集），取最小特征值对应的特征向量。

                        二者目标恰好相反。在分类问题中，有时先用PCA降维以去噪，但这里专门针对一类样本的散布矩阵，目的是压缩类内变化。

                        6. 完整算法步骤（与图片一致）

                        1. 对给定的单类样本集，计算协方差矩阵 C（类内散布矩阵）。

                        2. 计算 C 的所有特征值 λ1≤λ2≤⋯≤λn，并记对应的特征向量 u1,u2,…,un（单位化）。

                        3. 选择前 m 个最小特征值对应的特征向量 u1,u2,…,um，构造变换矩阵

                          A=[u1Tu2T⋮umT]m×n

                        4. 对任意原始样本 X，降维后得 X∗=AX（m 维列向量）。

                        7. 例题验证

                        已知：

                        X1=[11],X2=[22],X3=[31]

                        7.1 计算协方差矩阵（无偏）

                        均值：

                        μ=13[1+2+31+2+1]=[24/3]

                        偏差：

                        d1=[−1−1/3],d2=[02/3],d3=[1−1/3]

                        外积和：

                        ∑didiT=[11/31/31/9]+[0004/9]+[1−1/3−1/31/9]=[2006/9]=[2002/3]

                        除以 m−1=2：

                        C=[1001/3]

                        7.2 特征值与特征向量

                        λ1=1/3,u1=[01],λ2=1,u2=[10]

                        从小到大排列： λ1=1/3（最小）， λ2=1。

                        7.3 降到一维 ( m=1)

                        选择最小特征值对应的特征向量 u1=[0,1]T。构造 A=[0 1]。投影：

                        X1∗=0⋅1+1⋅1=1,X2∗=0⋅2+1⋅2=2,X3∗=0⋅3+1⋅1=1

                        结果：1,2,1，与图片答案一致。

                        8. 总结

                        • 基于类内散布矩阵的特征提取本质是最小化类内投影方差，因此选择协方差矩阵的最小特征值对应的特征向量。

                        • 数学上，该问题转化为求解特征方程 Cu=λu，最优投影方向即特征向量，投影方差即对应特征值。

                        • 降维时，取最小的若干个特征值对应的特征向量构成变换矩阵。

                        • 这与PCA（取最大特征值）目标不同，适用于需要压缩类内差异的场景（如某些分类预处理或数据压缩）。
